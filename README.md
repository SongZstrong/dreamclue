# Dreambook

这是一个基于 Next.js 16 的梦境记录与解梦检索项目，核心业务围绕以下几块展开：

- 梦境日志记录、编辑、删除、AI 分析
- Dreambook 知识库上传、分块、向量化、检索
- Stripe 订阅/一次性购买
- Better Auth 登录体系、后台用户管理、Newsletter/邮件通知
